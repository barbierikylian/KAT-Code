Introduction au HTML

Le HTML (HyperText Markup Language) est le langage de balisage standard pour créer des pages web. C'est la structure fondamentale de tout site web.

Qu'est-ce que le HTML ?

Le HTML n'est pas un langage de programmation, mais un langage de balisage qui définit la structure de votre contenu. Il utilise des balises pour organiser et présenter le contenu web.

<!DOCTYPE html> <html> <head> <title>Ma première page</title> </head> <body> <h1>Bonjour monde !</h1> <p>Ceci est un paragraphe.</p> </body> </html>

Résultat :

Bonjour monde !

Ceci est un paragraphe.

Les balises HTML

Les balises HTML sont les blocs de construction de toute page web. Voici quelques points importants :

  • Les balises sont généralement utilisées par paires : une balise ouvrante et une balise fermante
  • La balise fermante contient un slash (/) avant le nom de la balise
  • Certaines balises sont auto-fermantes (comme <img>)

Exercice pratique

Créez une structure HTML basique contenant :

  • Un titre principal
  • Deux paragraphes
  • Une liste à puces
