Introduction au HTML

Bienvenue dans votre première leçon ! Aujourd'hui, vous allez apprendre les bases du HTML.

Qu'est-ce que le HTML ?

Le HTML (HyperText Markup Language) est le langage de balisage, c'est à dire un langage qui utilise des balises pour définir les éléments d'un document, utilisé pour créer des pages web. Il sert à définir la structure et le contenu des sites internet.

Si vous voulez en savoir plus sur l'origine du language HTML, cliquez ici.

Structure de base d'une page HTML

Chaque page commence toujours par les mêmes balises. Voici à quoi ressemble une structure de code HTML basique :

  • <!DOCTYPE html> : Cette ligne est toujours la première d'un fichier HTML. Elle indique au navigateur internet (Chrome, Safari, Firefox, ...) que vous utilisez la dernière version du HTML (HTML5).
  • <html> : Cette balise est la racine de votre document HTML. Tout son contenu doit se trouver à l'intérieur de cette balise.
  • <head> : Cette section contient des informations sur la page, comme son titre, ses styles ou des scripts.
  • <title> : Définit le titre de la page qui s'affiche dans l'onglet de votre navigateur.
  • <body> : Contient tout le contenu visible de votre page, comme des titres, paragraphes, images, etc.
<!DOCTYPE html>
<html>
  <head>
    <title>Ma première page</title>
  </head>
  <body>
    <h1>Bonjour tout le monde !</h1>
    <p>Ceci est un paragraphe.</p>
  </body>
</html>

Résultat :


Bonjour tout le monde !

Ceci est un paragraphe.

Les balises HTML

Les balises HTML sont les blocs de construction d'une page web. Voici quelques points importants :

  • Les balises sont généralement utilisées par paires : une balise ouvrante et une balise fermante.
  • La balise fermante contient un slash (/) avant le nom de la balise (par exemple </title>).
  • Certaines balises sont auto-fermantes car elle ne nécessite pas de balise de fermeture (comme <img>).

Balises fondamentales

Voici quelques balises importantes :

  • <h1> : Affiche un titre principal. Chaque page devrait en contenir qu'un seul.
  • <h2> : Affiche un titre secondaire. Une page peut en contenir plusieurs. (Il existe d'autres balises de titre en fonction de son importance : <h3>, <h4>, ...).
  • <p> : Permet d'afficher un paragraphe de texte.
  • <a> : Créer un lien permettant de naviguer vers une autre page. La balise ouvrante contient le lien vers l'autre page web dans un de ses attributs.
  • <img> : Insére une image dans la page. La balise contient directement les informations comme la source de l'image, à l'intérieur.
  • <ul> et <li> : Utilisés ensemble pour créer des listes à puces.
  • <br> : Permet un retour à la ligne après un élément
<h1>Bienvenue sur ma page</h1>
<p>Ceci est un paragraphe d'exemple.</p>
<a href="https://google.com">Visitez ce lien</a><br>
<img src="image.jpg" alt="Exemple d'image">
<ul>
  <li>Élément de liste 1</li>
  <li>Élément de liste 2</li>
  <li>Élément de liste 3</li>
</ul>

Organisation du contenu avec HTML

Dans cette leçon, vous allez apprendre à comment structurer efficacement une page web en utilisant des balises adaptées, en améliorant sa lisibilité et son accessibilité.

Les conteneurs

Les conteneur sont des éléments qui servent à structurer et organiser le contenu d'un page web. Ils permettent de regrouper des éléments afin de les manipuler plus facilement.

Le conteneur générique <div>

La balise <div> est un conteneur générique de type bloc utilisé pour structurer une page web. Elle permet la plupart du temps de regrouper plusieurs éléments sans leur attribuer de signification particulière.

Les conteneurs sémantiques

Contrairement au div, les conteneurs sémantiques donnent un sens au contenu qu'ils englobent, facilitant le référencement.

  • <header> & <footer> : Respectivement l'en-tête et le pied de page d'une page.
  • <main> : Enveloppe le contenu principal de la page, généralement entre le header et le footer.
  • <nav> : Spécifique aux menus de navigation.
  • <aside> : Contient des informations complémentaires, comme des liens ou un encart publicitaire.
  • <article> : Contient un article indépendant.
  • <section> : Regroupe un ensemble de contenus, souvent par thème.

Les formulaires en HTML

Les formulaires sont un élément clé des sites web : ils permettent d’interagir avec les visiteurs, que ce soit pour recueillir des informations, envoyer un message ou s’inscrire à un service. Dans ce chapitre, vous découvrirez comment créer des formulaires simples et efficaces pour rendre votre site plus interactif.

La balise <form>

La balise <form> est un conteneur sémantique utilisée pour créer un formulaire permettant aux utilisateurs de saisir des informations.

<form>
  <!-- Contenu du formulaire ici -->
</form>

Les champs de saisie <input>

La balise <input> permet de créer des champs de saisie de texte.
La façon dont l'élément <input> fonctionne dépend de son type <input type="...">. Voici les types les plus courants :

  • type="text" : champ de texte.
  • type="email" : champ pour les emails.
  • type="date" : champ pour les dates.
  • type="password" : champ pour les mots de passe.
  • type="checkbox" : champ pour les cases à cocher.

Il est possible d'ajouter un placeholder à la balise qui est un texte d'exemple à l'intérieur du champ, visible tant qu'aucune donnée n'a été saisie.
Nous pouvons aussi rendre ce champ obligatoire pour l'envoie du formulaire grâce à l'attribut required.

<input type="text" required>
<input type="email" placeholder="Votre email">
<input type="date">
<input type="password" placeholder="Votre mot de passe">
<input type="checkbox">

Introduction au CSS

Jusqu'ici, vous avez créé et structuré une page web avec HTML, mais sans style. Avec le CSS, vous pourrez personnaliser les couleurs, tailles et mises en page. Ce chapitre vous apprendra les bases pour styliser votre site.

Qu'est-ce que le CSS ?

Le CSS (Cascading Style Sheets) est un langage utilisé pour décrire l'apparence des pages web. Il permet de définir les couleurs, les polices d'écritures, les positions des éléments et bien plus encore.

Comment lier un fichier CSS à une page HTML ?

Il existe trois manières d'ajouter du CSS :

  • CSS en ligne : ajouté directement dans l'attribut style d'une balise. Nous l'utiliserons au début.
  • CSS interne : ajouté dans une balise <style> dans le <head>de la page.
  • CSS externe : stocké dans un fichier séparé et lié avec la balise <link>. C'est la méthode la plus recommandée pour les sites web.

Sélection par classe

Une classe permet de styliser plusieurs éléments en même temps. Pour ajouter une classe à un élément, on utilise l’attribut class dans la balise HTML :

<p class="exemple">Ce texte a une classe !</p>

Puis en CSS, Utilisez un point (.) pour cibler la classe.

<head>

    .exemple {

    }

</head>
<body>

  <p>Ce texte n'a pas la classe exemple, sa taille n'est donc pas modifiée.</p>
  <p class="exemple">La taille de la police d'écriture de ce texte aussi plus grand, grâce à la classe exemple.</p>

Sélection par ID

Un ID permet de styliser un seul élément unique. Pour ajouter un ID à un élément, on utilise l’attribut id dans la balise HTML :

<p id="unique">Ce texte a un ID !</p>

Puis en CSS, utilisez un dièse (#) pour cibler l’ID.

<head>
  <style>
    #unique {
      color: red;
    }
  </style>
</head>
<body>

  <p>Ce texte n'a pas d'ID unique, il reste en couleur par défaut.</p>
</body>

Résultat :


Ce texte est en rouge.

Marges et Espacements en CSS

Ce chapitre est intermédiaire, car il vous aidera à mieux comprendre comment organiser vos éléments sur une page. Après avoir appris les bases, il est temps d'explorer comment contrôler les espaces entre et à l'intérieur des éléments pour améliorer la lisibilité et l'esthétique de vos éléments.

Les propriétes de marges et d'espacements

Les propriétés margin(marge) et padding(espacement) permettent de gérer l’espace autour et à l’intérieur des éléments. Elles jouent un rôle clé dans l’organisation et l’apparence des pages web.

Dans la représentation ci-dessus, la zone margin se trouve en orange, la zone border en jaune, la zone padding en vert et le contenu HTML en bleu :

  • La zone border permet de créer une bordure visible à l'élément, comme vous l'avez vu au chapitre précédent.
  • La zone padding permet de créer un espace entre l'élément HTML visible et sa bordure.
  • La zone margin permet de créer un espace entre la bordure de l'élément et les autres éléments de la page.

Lorsque vous ajoutez un fond de couleur et un padding, celui-ci s’étend avec l’arrière-plan, rendant la marge intérieure visible. En revanche, margin reste invisible et n’agrandit pas l’élément.

🔹 La marge (margin)

La propriété margin définit l’espace extérieur d’un élément, c’est-à-dire l’écart entre cet élément et les autres autour de lui. Vous pouvez l'ajouter de différente façon :

.marge {
  margin: 10px
}

.marge2 {
  margin: 10px 0px;
}
.marge3 {
  margin: 20px 5px 7px 2px;
}
  • marge→ L'élément aura une marge de 10 pixels sur tous les côtés.
  • marge2→ L'élément aura une marge de 10 pixels en haut et en bas, mais aucune marge à droite et à gauche.
  • marge3→ L'élément aura une marge différente pour chaque côté (haut, droite, bas, gauche)

Mise en page avec CSS

Dans cette leçon, nous allons explorer différentes techniques de mise en page en CSS.

1. Mode d'affichage (display)

En CSS, la propriété display définit comment un élément doit être affiché dans la structure générale de la page.

1.1 - display: blockvsdisplay: inline


  • display: block; : Les éléments prennent toute la largeur disponible et force un retour à la ligne.
  • display: inline; : Les éléments ne prennent que la place nécessaire et reste sur la même ligne que les autres éléments inline.
<div style="display: block;">Bloc 1</div>
<p>Je suis un <span style="display: inline;">texte en ligne</span> dans un paragraphe</p>

Résultat :


Bloc 1

2. Utilisation de Flexbox

Flexbox est un outil puissant qui permet d'organiser facilement des éléments sur une page, que ce soit horizontalement ou verticalement.

2.1 Propriétés principales


La zone d'un document qui utilise Flexbox est appelée un conteneur flexible. Pour le créer, on doit donner à un élément la propriété display: flex; (pour les éléments de type bloc) ou display: inline-flex; (pour les éléments de type inline). Dès que cela est fait, tous les éléments à l'intérieur deviennent des éléments flexibles.

  • flex: Le conteneur prend toute la largeur disponible.
  • inline-flex: Le conteneur prend seulement la place nécessaire, comme un élément inline.

Un élément flex est très utile pour organiser les éléments de manière fluide et responsive. Il permet un meilleur contrôle de l'alignement et de la répartition de l'espace sans avoir à utiliser énormement de ligne en CSS.

<div style="display: flex;">
  <div>Élément 1</div>
  <div>Élément 2</div>
</div>

<p>Je suis un
  <span style="display: inline-flex;">
    Élément inline-flex
  </span>
qui peut être stylisé.</p>

Élément 2

Dans le second, align-items: center; les centre verticalement malgré leurs tailles différentes.

Ces deux attributs peuvent avoir d'autres paramètres :

  • justify-content: contrôle l'alignement horizontalement des éléments flexibles.

    • flex-start : aligne les éléments au début du conteneur (par défaut).
    • flex-end : aligne les éléments à la fin du conteneur.
    • center : centre les éléments horizontalement.
    • space-between : répartit l'espace entre les éléments.
    • space-around : ajoute un espace égal autour de chaque élément.
    • space-evenly : distribue l'espace de manière égale entre et autour des éléments.

  • align-items: contrôle l'alignement vertical des éléments flexibles.

    • flex-start : aligne les éléments en haut du conteneur.
    • flex-end : aligne les éléments en bas du conteneur.
    • center : centre les éléments verticalement.
    • stretch : étire les éléments pour remplir l'espace disponible (par défaut).
    • baseline : aligne les éléments sur leur ligne de base du texte.

2.3 Les axes en Flexbox


  • Axe principal : défini par flex-direction (row ou column).
  • Axe secondaire : défini par align-items.

flex-direction: définit l'axe principal sur lequel les éléments seront placés.

  • row(ligne) : aligne les éléments au début du conteneur (par défaut).

  • column(colonne) : ajoute un espace égal autour de chaque élément.
  • column-reverse : distribue l'espace de manière égale entre et autour des éléments.

3. Positionnement avec CSS

Les éléments peuvent être positionnés de différentes façons :

  • static : Position par défaut des éléments HTML. Ils sont placés selon la structure normale du document et ne réagissent pas aux propriétés top, right, leftou bottom.
  • relative : Position par rapport à sa position normale.
  • absolute : Position par rapport à son premier parent positionné.
  • fixed : Position fixe par rapport à la fenêtre.
  • sticky : Position fixe par rapport à la fenêtre.

Les propriétés top, right, leftet bottom permettent de déplacer un élément en fonction de son type de positionnement (relative, absolute, fixed, sticky).

.position {
  position: absolute; // Peut être relative, absolute, fixed, sticky
  top: 50px; // Déplace l'élément de 50px vers le bas
  left: 100px; // Déplace l'élément de 100px vers la droite
}

Transitions fluides en CSS

La propriété transition permet de créer une animation fluide entre deux états CSS.

Les paramètres des transitions

Syntaxe :

transition: propriété durée timing-function delay;
  • Propriété→ La propriété que l'on veut animer (exemple: background-color)
  • Durée→ Temps de l'animation (ex: 0.5s)
  • Timing-function→ Détermine la vitesse (ease, ease-in, ease-out, linear...)
  • Delay(optionnel)→ Temps d’attente avant de commencer l’animation.

Différentes valeurs de timing-function

  • linear : L'animation avance à une vitesse constante.
  • ease : L'animation commence lentement, accélère puis ralentit à la fin.
  • ease-in : L'animation commence lentement puis accélère.
  • ease-out : L'animation démarre vite et ralentit à la fin.
  • ease-in-out : L'animation commence et finit lentement.
.button {
  background-color: blue;
  color: white;
  padding: 10px 20px;
  border: none;
  transition: background-color 0.5s ease;
}
.button:hover {
  background-color: red;
}

Résultat :


📌 Explication : Quand la souris passe sur le bouton, la couleur change progressivement en 0.5s.

Animer un élément avec @keyframes

La propriété animation

Syntaxe :

animation: nom-animation durée timing-function delay iteration-count direction fill-mode;
Paramètre Description
nom-animation Le nom de l'animation définie avec @keyframes.
durée Temps que met l’animation pour se jouer (ex: 2s, 500ms).
timing-function Détermine la courbe d'accélération (ease, linear, etc.).
delay (optionnel) Temps d’attente avant le début de l’animation.
iteration-count (optionnel) Nombre de répétitions de l’animation (1, infinite, etc.).
direction (optionnel) Définit le sens de l’animation (normal, reverse, alternate, etc.).
fill-mode (optionnel) Contrôle l’apparence de l’élément avant/après l’animation (none, forwards, etc.).

Les animations permettent de créer des effets plus avancés qu’une simple transition.

@keyframes permet de créer une animation complète, en définissant différentes étapes.

Structure d’un @keyframes :

@keyframes nom-animation {
  0% {
    propriété: valeur de départ;
  }
  50% {
    propriété: valeur intermédiaire;
  }
  100% {
    propriété: valeur finale;
  }
}

@keyframes nom-animation {
  from {
    propriété: valeur de départ;
  }
  to {
    propriété: valeur finale;
  }
}


  • from ou 0%→ Début de l’animation.
  • 50%→ Milieu de l’animation.
  • to ou 100%→ Fin de l’animation.

Par exemple, pour créer un effet de changer de la couleur de fond d'un texte, nous pouvons utiliser l'animation keyframes comme ceci :

.paragraphe-anime {
  animation: couleur 5s infinite ease-in-out
}

@keyframes couleur {
  0% {
    background-color: red;
  }
  50% {
    background-color: green;
  }
  100% {
    background-color: blue;
  }
}

Résultat :


La couleur de fond de ce texte est animée

📌 Explication : La couleur de fond de ce texte est intialement rouge, mais devient vert puis bleu. L'animation se répète à l'infini grâce à l'attribut infinite.

Comprendre les media queries

Les media queries permettent d'adapter le style des éléments en fonction de la taille de l'écran.

@media (max-width: 600px) {

  • On peut aussi utiliser min-width pour appliquer des styles à partir d'une certaine largeur d'écran.
.button {
  background-color: yellow;
  color: black;
}

@media (max-width: 600px) {
  .button {
    background-color: purple;
  color: white;
  }
}

🖥️ C'est quoi le Back-End ?

Si tu souhaite vraiment aller plus loin et que tu veux un site où les utilisateurs peuvent se connecter, poster des messages ou enregistrer des données ? C'est là qu'intervient le Back-End !

Le Back-End, c'est la partie cachée d'un site web qui permet de stocker et gérer des informations.

  • Son rôle : gérer la base de données, les connexions des utilisateurs et tout ce qui se passe "dans l'ombre" d'un site.
  • Langages utilisés : PHP, JavaScript (Node.js), Python (Django, Flask), Ruby.
  • Bases de données : MySQL, PostgreSQL, MongoDB.
