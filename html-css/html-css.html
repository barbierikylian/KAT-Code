Introduction au HTML

Bienvenue dans votre première leçon ! Aujourd'hui, vous allez apprendre les bases du HTML.

Qu'est-ce que le HTML ?

Le HTML (HyperText Markup Language) est le langage de balisage, c'est à dire un langage qui utilise des balises pour définir les éléments d'un document, utilisé pour créer des pages web. Il sert à définir la structure et le contenu des sites internet.

Si vous voulez en savoir plus sur l'origine du language HTML, cliquez ici.

Structure de base d'une page HTML

Chaque page commence toujours par les mêmes balises. Voici à quoi ressemble une structure de code HTML basique :

  • <!DOCTYPE html> : Cette ligne est toujours la première d'un fichier HTML. Elle indique au navigateur internet (Chrome, Safari, Firefox, ...) que vous utilisez la dernière version du HTML (HTML5).
  • <html> : Cette balise est la racine de votre document HTML. Tout son contenu doit se trouver à l'intérieur de cette balise.
  • <head> : Cette section contient des informations sur la page, comme son titre, ses styles ou des scripts.
  • <title> : Définit le titre de la page qui s'affiche dans l'onglet de votre navigateur.
  • <body> : Contient tout le contenu visible de votre page, comme des titres, paragraphes, images, etc.
<!DOCTYPE html>
<html>
  <head>
    <title>Ma première page</title>
  </head>
  <body>
    <h1>Bonjour tout le monde !</h1>
    <p>Ceci est un paragraphe.</p>
  </body>
</html>

Résultat :

Bonjour tout le monde !

Ceci est un paragraphe.

Les balises HTML

Les balises HTML sont les blocs de construction d'une page web. Voici quelques points importants :

  • Les balises sont généralement utilisées par paires : une balise ouvrante et une balise fermante.
  • La balise fermante contient un slash (/) avant le nom de la balise (par exemple </title>).
  • Certaines balises sont auto-fermantes car elle ne nécessite pas de balise de fermeture (comme <img>).

Balises fondamentales

Voici quelques balises importantes :

  • <h1> : Affiche un titre principal. Chaque page devrait en contenir qu'un seul.
  • <h2> : Affiche un titre secondaire. Une page peut en contenir plusieurs. (Il existe d'autres balises de titre en fonction de son importance : <h3>, <h4>, ...).
  • <p> : Permet d'afficher un paragraphe de texte.
  • <a> : Créer un lien permettant de naviguer vers une autre page. La balise ouvrante contient le lien vers l'autre page web dans un de ses attributs.
  • <img> : Insére une image dans la page. La balise contient directement les informations comme la source de l'image, à l'intérieur.
  • <ul> et <li> : Utilisés ensemble pour créer des listes à puces.
  • <br> : Permet un retour à la ligne après un élément
<h1>Bienvenue sur ma page</h1>
<p>Ceci est un paragraphe d'exemple.</p>
<a href="https://google.com">Visitez ce lien</a><br>
<img src="image.jpg" alt="Exemple d'image">
<ul>
  <li>Élément de liste 1</li>
  <li>Élément de liste 2</li>
  <li>Élément de liste 3</li>
</ul>

Résultat :

Bienvenue sur ma page

Ceci est un paragraphe d'exemple.

Visitez ce lien
  • Élément de liste 1
  • Élément de liste 2
  • Élément de liste 3

Organisation du contenu avec HTML

Dans cette leçon, vous allez apprendre à comment structurer efficacement une page web en utilisant des balises adaptées, en améliorant sa lisibilité et son accessibilité.

Les conteneurs

Les conteneur sont des éléments qui servent à structurer et organiser le contenu d'un page web. Ils permettent de regrouper des éléments afin de les manipuler plus facilement.

Le conteneur générique <div>

La balise <div> est un conteneur générique de type bloc utilisé pour structurer une page web. Elle permet la plupart du temps de regrouper plusieurs éléments sans leur attribuer de signification particulière.

  • <div> : Conteneur de type bloc utilisé pour regrouper plusieurs éléments.

</div>

  <p>Il regroupe plusieurs éléments</p>
  <a href="google.fr">Faites une recherche Google</a>
</div>

Les conteneurs sémantiques

Contrairement au div, les conteneurs sémantiques donnent un sens au contenu qu'ils englobent, facilitant le référencement.

  • <header> & <footer> : Respectivement l'en-tête et le pied de page d'une page.
  • <main> : Enveloppe le contenu principal de la page, généralement entre le header et le footer.
  • <nav> : Spécifique aux menus de navigation.
  • <aside> : Contient des informations complémentaires, comme des liens ou un encart publicitaire.
  • <article> : Contient un article indépendant.
  • <section> : Regroupe un ensemble de contenus, souvent par thème.

Le code :

<header>HEADER</header>
<nav>NAVIGATION</nav>
<main>
  <aside>SIDEBAR<aside>
  <div>
    <div>MAIN CONTENT</div>
    <article>ARTICLE</article>
    <section>SECTION</section>
  </div>
</main>
<footer>FOOTER</footer>

Les formulaires en HTML

Les formulaires sont un élément clé des sites web : ils permettent d’interagir avec les visiteurs, que ce soit pour recueillir des informations, envoyer un message ou s’inscrire à un service. Dans ce chapitre, vous découvrirez comment créer des formulaires simples et efficaces pour rendre votre site plus interactif.

La balise <form>

La balise <form> est un conteneur sémantique utilisée pour créer un formulaire permettant aux utilisateurs de saisir des informations.

<form>
  <!-- Contenu du formulaire ici -->
</form>

Les champs de saisie <input>

La balise <input> permet de créer des champs de saisie de texte.
La façon dont l'élément <input> fonctionne dépend de son type <input type="...">. Voici les types les plus courants :

  • type="text" : champ de texte.
  • type="email" : champ pour les emails.
  • type="date" : champ pour les dates.
  • type="password" : champ pour les mots de passe.
  • type="checkbox" : champ pour les cases à cocher.

Il est possible d'ajouter un placeholder à la balise qui est un texte d'exemple à l'intérieur du champ, visible tant qu'aucune donnée n'a été saisie.
Nous pouvons aussi rendre ce champ obligatoire pour l'envoie du formulaire grâce à l'attribut required.

<input type="text" required>
<input type="email" placeholder="Votre email">
<input type="date">
<input type="password" placeholder="Votre mot de passe">
<input type="checkbox">

Introduction au CSS

Dans ce chapitre, vous allez apprendre les bases du CSS et comment styliser vos pages web.

Qu'est-ce que le CSS ?

Le CSS (Cascading Style Sheets) est un langage utilisé pour décrire l'apparence des pages web. Il permet de définir les couleurs, les polices, les marges, et bien plus encore.

Comment lier une page HTML à une feuille CSS ?

Il existe trois manières d'ajouter du CSS :

  • CSS en ligne : ajouté directement dans l'attribut style d'une balise.
  • CSS interne : ajouté dans une balise <style> dans le <head>.
  • CSS externe : stocké dans un fichier séparé et lié avec <link>.
<head>
  <link rel="stylesheet" href="styles.css">
</head>

Résultat :

Le fichier "styles.css" est maintenant appliqué à votre page.

Sélecteurs CSS

Les sélecteurs permettent de cibler des éléments HTML pour les styliser.

Sélection par balise

Applique un style à toutes les balises du même type :

p { color: blue; }

Résultat :

Ce texte est en bleu.

Sélection par classe

Utilise un point (.) pour sélectionner des éléments avec une classe spécifique :

.important { font-weight: bold; }

Résultat :

Ce texte est en gras.

Sélection par ID

Utilise un dièse (#) pour cibler un élément unique :

#titre { font-size: 24px; }

Résultat :

Ce titre est plus grand.

Modifier l’apparence du texte

Changer la couleur

Utilisez la propriété color pour modifier la couleur du texte :

p { color: red; }

Résultat :

Ce texte est rouge.

Modifier la taille

Utilisez font-size pour changer la taille du texte :

h1 { font-size: 32px; }

Résultat :

Titre agrandi

Modifier la police

Utilisez font-family pour changer la police d’écriture :

body { font-family: Arial, sans-serif; }

Mise en page avec CSS

Dans cette leçon, nous allons explorer différentes techniques de mise en page en CSS.

Display : block vs inline

En CSS, les éléments peuvent être affichés de différentes manières :

  • display: block; : Les éléments prennent toute la largeur disponible.
  • display: inline; : Les éléments s'affichent à la suite des autres sans retour à la ligne.
<div style="display: block; background-color: lightblue;">Bloc 1</div>
        <span style="display: inline; background-color: lightgreen;">Texte en ligne</span>

Résultat :

Bloc 1
Texte en ligne

Utilisation de Flexbox

Flexbox permet d'organiser les éléments horizontalement ou verticalement.

<div style="display: flex; gap: 10px;">
            <div style="background-color: lightcoral; padding: 10px;">Élément 1</div>
            <div style="background-color: lightseagreen; padding: 10px;">Élément 2</div>
        </div>

Résultat :

Élément 1
Élément 2

Utilisation de CSS Grid

Grid permet une mise en page plus avancée.

<div style="display: grid; grid-template-columns: 1fr 1fr; gap: 10px;">
            <div style="background-color: lightpink; padding: 10px;">Colonne 1</div>
            <div style="background-color: lightgoldenrodyellow; padding: 10px;">Colonne 2</div>
        </div>

Résultat :

Colonne 1
Colonne 2

Positionnement avec CSS

Les éléments peuvent être positionnés de différentes façons :

  • relative : Position par rapport à sa position normale.
  • absolute : Position par rapport à son premier ancêtre positionné.
  • fixed : Position fixe par rapport à la fenêtre.

Création d'une barre de navigation

Voici un exemple de barre de navigation stylisée avec du CSS :

<nav style="background-color: navy; padding: 10px;">
            <a href="#" style="color: white; margin-right: 10px;">Accueil</a>
            <a href="#" style="color: white;">Contact</a>
        </nav>

Résultat :

Accueil Contact

Responsive Design

Le CSS permet d'adapter les pages aux écrans :

@media (max-width: 600px) {
            body {
                background-color: lightgray;
            }
        }

Chapitre 5 : Animation et Intéractions en CSS

Dans cette leçon, nous allons explorer comment animer et rendre nos éléments interactifs en CSS.

1. Introduction aux animations

Les animations en CSS permettent de donner vie aux éléments en appliquant des changements progressifs.

Les transitions

Les transitions permettent de créer des effets de changement progressifs entre deux états CSS.

    .button {
        background-color: blue;
        color: white;
        padding: 10px 20px;
        border: none;
        transition: background-color 0.5s ease;
    }
    .button:hover {
        background-color: red;
    }

Résultat :

Les animations avec @keyframes

Avec @keyframes, on peut créer des animations plus complexes.

    @keyframes bounce {
        0%, 100% {
            transform: translateY(0);
        }
        50% {
            transform: translateY(-20px);
        }
    }
    .animated-box {
        width: 100px;
        height: 100px;
        background-color: green;
        animation: bounce 1s infinite;
    }

2. Effets interactifs

Changement de couleur au survol

Avec :hover, on peut modifier l’apparence d’un élément lorsqu'on passe la souris dessus.

    p:hover {

    }
